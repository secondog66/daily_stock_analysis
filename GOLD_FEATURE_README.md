# 黄金分析功能

## 概述

本项目已成功集成黄金价格分析功能，使您能够像分析股票一样分析黄金价格。黄金分析功能完全兼容现有的股票分析系统，具备相同的分析能力和功能。

## 新增功能

- **黄金价格数据获取**：支持GOLD、GC=F、XAUUSD=X、XAU等多种黄金代码
- **技术指标分析**：与股票相同的MA5、MA10、MA20、量比、涨跌幅等技术指标
- **实时行情获取**：支持黄金实时价格监控
- **历史数据分析**：支持黄金历史价格趋势分析
- **策略引擎**：支持专业的黄金分析策略
- **通知推送**：集成到现有的企业微信、飞书、邮件等推送系统

## 使用方法

### 1. 数据获取

```python
from data_provider.base import DataFetcherManager

# 创建数据管理器
manager = DataFetcherManager()

# 获取黄金历史数据（与股票方式相同）
df, source = manager.get_daily_data('GOLD', days=30)

# 获取黄金实时行情（与股票方式相同）
quote = manager.get_realtime_quote('GOLD')
```

### 2. 在自选股中添加黄金

在配置文件中添加黄金到自选股列表：

```
STOCK_LIST = '600519,000001,GOLD'  # 同时分析股票和黄金
```

### 3. 支持的黄金代码

- `GOLD` - 通用黄金代码
- `GC=F` - COMEX黄金期货
- `XAUUSD=X` - XAU/USD现货黄金
- `XAU` - XAU黄金

### 数据标准化

黄金数据使用与股票完全相同的标准格式：
- `date`: 日期
- `open`: 开盘价
- `high`: 最高价
- `low`: 最低价
- `close`: 收盘价
- `volume`: 成交量
- `amount`: 成交额
- `pct_chg`: 涨跌幅

### 技术指标

系统自动计算以下技术指标：
- MA5、MA10、MA20：移动平均线
- volume_ratio：量比
- 以及其他高级技术指标

## 数据源可靠性

- **主要数据源**：Yahoo Finance（GC=F, XAUUSD=X）
- **备用数据源**：多个备用API和故障切换机制
- **API限制处理**：当遇到Rate Limit时自动切换数据源
- **离线模式**：支持缓存机制减少API调用

## 风险提示

1. 黄金价格受国际政治经济因素影响较大
2. 数据可能存在延迟，实时性不如股票
3. 部分时间段可能无法获取实时数据
4. 技术指标分析仅作参考，不构成投资建议

## 故障排除

### 常见问题

**Q: 获取黄金历史数据失败？**
A: 可能是Yahoo Finance API限制，请稍后再试，系统会自动切换数据源。

**Q: 实时行情获取不到？**
A: 检查API是否正常，系统会自动使用备用数据源。
